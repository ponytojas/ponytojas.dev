# Personal Website (Astro)

This project is now built with [Astro](https://astro.build), React components, and Tailwind CSS v4.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
pnpm build
pnpm start
```

## Tech Stack

- Astro
- React 19
- Tailwind CSS v4
- MDX (via Vite `@mdx-js/rollup`)
